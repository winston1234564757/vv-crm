Що означає Mobile First для CRM?

Потрібно зрозуміти глибину:

A

Повноцінна мобільна версія

Усі функції доступні на телефоні: створення продажу, ремонту, перегляд клієнтів. Адаптивний дизайн, touch-орієнтовані елементи.

B

Tablet-friendly + responsive

Основний сценарій — планшет біля прилавка + ноутбук. Телефон — для швидких переглядів, але не для важкої роботи.

C

Mobile-First підхід у CSS

Будуємо на mobile-first сітках та компонентах, але основна ціль — десктоп/планшет під час роботи. Mobile як бонус.

Вибери варіант або розкажи детальніше, який mobile-сценарій важливий.
